# Resume Builder V2.4 Acceptance Checklist

Date: 2026-07-11

## Automated Verification

```bash
npm test
```

预期全部通过，无失败、取消或跳过，并覆盖：

- [ ] 默认值、范围、步长、枚举、未知 key 和旧 YAML 兼容。
- [ ] 间距锚点、插值、页边距、字号比例和候选顺序。
- [ ] 自动模式首个可用候选、固定模式单候选和精确溢出量。
- [ ] 草稿与正式生成共享候选语义，旧异步响应不能覆盖新状态。
- [ ] 溢出允许保存但阻止生成，恢复单页后生成重新可用。
- [ ] 每份简历独立设置，切换、复制、备份、导入导出和恢复不回归。
- [ ] 标准 A4、单页 PDF、PNG、HTML 和点击定位标记不回归。
- [ ] 隐私扫描、路径边界和 GitHub Actions CI 通过。

## Safe Manual Setup

- [ ] 只使用虚构姓名、占位联系方式、虚构经历和测试照片。
- [ ] 创建仓库外临时数据根，不读取或复制个人运行数据：

```bash
manual_root="$(mktemp -d)/resume-builder-data"
RESUME_BUILDER_DATA_DIR="$manual_root" npm run editor
```

- [ ] 验收产生的 YAML、照片、PDF、PNG、HTML 和 ZIP 不加入 Git。

## Layout Controls

- [ ] 顶层入口显示 `内容编辑` 和 `排版设置`。
- [ ] 自动/固定选择、正文字号、行高、内容间距、页边距、恢复默认和模块顺序完整可用。
- [ ] 字号仅为 `10.2-11.2pt`，行高仅为 `1.25-1.42`，间距仅为整数 `0-100`。
- [ ] 滑杆、减号、加号和 segmented controls 可用键盘操作，焦点可见；达到边界时对应步进按钮禁用。
- [ ] `紧凑 / 较紧 / 标准 / 宽松` 是锚点说明，不显示为百分比。
- [ ] `恢复默认` 只恢复排版参数，不改变姓名、联系方式、经历、技能、项目或照片。
- [ ] 模块上移/下移继续改变草稿顺序，并随当前简历保存。

## Draft Fitting

- [ ] 自动模式先使用首选值；首选可放下时不继续压缩。
- [ ] 内容稍长时依次压缩间距、页边距、行高和字号，且不低于硬下界。
- [ ] 状态栏显示实际生效的模式、字号、行距、紧凑度和边距。
- [ ] 候选可放下时显示 `A4 单页`；测量前显示 `A4 待测量`。
- [ ] 所有候选均失败时显示 `超出 A4 Npx`，没有文字裁切或错误的单页承诺。
- [ ] 连续快速修改时只有最后一份草稿和对应排版状态生效。

## Save And Generate Matrix

- [ ] 草稿溢出且未保存时，`保存` 可用，`生成 PDF` 不可用。
- [ ] 保存溢出 YAML 后，保存成功且备份存在，`生成 PDF` 仍不可用。
- [ ] 调整到可放入 A4 并保存后，`生成 PDF` 恢复可用。
- [ ] 固定模式不自动改变任何排版值；固定参数溢出时正式生成拒绝更新输出。
- [ ] 自动模式正式生成的最终参数与状态栏更新为服务端权威结果。
- [ ] 生成失败或溢出时，上一份正式 PDF、PNG 和 HTML 保持不变。

## Multi-Resume And Compatibility

- [ ] 两份简历可保存不同的模式、字号、行高、间距、边距和 section 顺序。
- [ ] 切换简历后显示该简历自己的设置，不继承上一份临时状态。
- [ ] 复制简历保留内容和排版设置，但不复制备份或生成输出。
- [ ] 仅含 `layout.sectionOrder` 的旧 YAML 可打开、预览、保存并生成。
- [ ] 数据包导出、导入和历史恢复后设置保持一致。

## Visual And Privacy Checks

- [ ] 宽屏和窄屏视口中，右侧控制无横向溢出，底部操作栏保持可见。
- [ ] 左侧 A4 完整显示；排版控件不改变点击定位与蓝色选择反馈。
- [ ] 自动和固定生成的 PDF 都是标准 A4 单页，照片仍在固定位置。
- [ ] `git diff --check`、`npm run privacy:check` 和 `git status --short` 不出现真实数据、绝对路径或临时验收文件。
